# BirthdaySite_Vanila_01
Birthday Celebration Web Page

Summary
- **What:** Mobile-first, celebratory photo gallery with guest book and slideshow support.
- **Purpose:** Display family and work photos during a birthday event, allow guests to leave wishes, and run a TV slideshow.

Key features
- Responsive gallery with thumbnails and full-screen viewer
- Manual swipe navigation for touch devices
- Automatic slideshow mode for TV displays
- Guest book for wishes/comments
- Optional background music and QR-code access
- Image optimization and lazy-loading for performance

Quick start (local)
1. Open `index.html` in a browser to preview locally.
2. Or run a simple local server from the project folder:
     - PowerShell / Python:
         ```powershell
         cd "C:\Users\jurit\.vscode\Jurim90\BirthdaySite_Vanila_01"
         python -m http.server 8000
         ```
     - Then open `http://localhost:8000` in your browser.
